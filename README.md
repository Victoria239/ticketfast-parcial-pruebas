# TicketFast - Parcial Pruebas de Software

Proyecto desarrollado para el examen parcial de Pruebas de Integracion, Sistema y E2E.

## Tecnologias

- Python
- FastAPI
- SQLAlchemy
- PostgreSQL
- Docker Compose
- Pytest
- HTTPX
- Playwright

## Requisitos del parcial cubiertos

1. Infraestructura de pruebas con docker-compose.test.yml
2. Prueba de integracion de API con acceso a base de datos
3. Prueba de sistema usando HTTP real con httpx
4. Prueba E2E de frontend usando Playwright
5. Repositorio con minimo 10 commits
